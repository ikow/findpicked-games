How to Play

  • Pencil: Tap or drag on the canvas to paint pixels with the selected color.
  • Fill: Tap any pixel to flood-fill all connected pixels of the same color.
  • Eraser: Removes pixels back to the transparent checker pattern.
  • Palette: Pick from 16 preset colors or use the custom color picker for any shade.
  • Grid Size: Switch between 8×8, 16×16, 24×24, and 32×32 — changing size starts a new canvas.
  • Undo: Step back through your last 50 actions.
  • Save: Stores your drawing in the browser so it persists on refresh.
  • Export PNG: Downloads your art as a high-resolution PNG file (10× scale, no grid).
