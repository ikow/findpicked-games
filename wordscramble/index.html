Rearrange the letters
Rearrange scrambled letters to form hidden words — beat the clock for a high score.
  • Goal: Tap the scrambled letters to spell the hidden English word before time runs out.
  • Play: Tap a letter tile to drop it into the next empty slot. Tap a filled slot to send the letter back.
  • Auto-check: When every slot is filled, your guess is checked instantly. Wrong answers shake — fix and try again.
  • Timer: You have one continuous clock (2 minutes on Easy, 2:30 on Medium, 3 minutes on Hard). Keep solving for as long as it lasts.
  • Streak bonus: Solving several words in a row adds a multiplier to each new solve — build long streaks for big scores.
  • Hint: Reveals one correct letter locked in place. You get 3 hints per game — using one halves that word's score.
  • Shuffle: Re-arrange the remaining tiles if you're stuck. Free, any time.
  • Skip: Give up on the current word — no points, breaks your streak.
  • Difficulty: Easy = 4–5 letters, Medium = 6–7, Hard = 8+. Choose a level, then tap New Game.
  • Keyboard: Type letters to place them, Backspace to remove the last placed letter.
