How to Play

  1. Fill every white cell with a digit from 1 to 9.
  2. Dark clue cells show sums: the number in the bottom-left is the target for the across run to the right; the number in the top-right is the target for the down run below.
  3. Digits in a single run must be unique — no repeats in any across or down sum group.
  4. Tap a white cell to select it, then tap a number (or press 1–9 on a keyboard). Use Erase or Backspace to clear.
  5. Mistakes highlight in red. Fill every cell correctly to win — faster solves earn more points.
